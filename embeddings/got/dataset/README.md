# Game of Thrones text data (local only)

The notebooks in `embeddings/got/` expect five text files named `001ssb.txt` through `005ssb.txt` in this directory.

These files are **not** included in the repository (copyright). Place your own copies here locally before running `lstm.ipynb` or `word2vec.ipynb`.

`character-deaths.csv` is included and can be used independently.
